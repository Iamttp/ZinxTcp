| 2      |   服务器端告知客户端玩家的pos（上线）                  |   syncPos  | |

| 4      |   服务器端告知客户端玩家的ID（下线）                   |   syncUnPid  | |
| 201    |   客户端发送移动请求和位置信息                        |   moveRouter.Handle | |
